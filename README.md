QtAV
==============

QtAV is a media playing library based on Qt and FFmpeg.

Build
------
  BUILD_DIR=/your/build/dir qmake

  or  qmake   (BUILD_DIR is $PWD/out)

  or  cd build_dir ; qmake /path/to/QtAV.pro

  make -j4

  The binaries is in $BUILD_DIR/bin
